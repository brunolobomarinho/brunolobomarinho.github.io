Ola hoje vou deixar uma dica rápida, se você roda seus projetos com maven na linha de comando, para testar um teste especifico rode.

Classe se chama Pagamento

mvn '-Dtest=br.com.uol.upbc.it.be.P*' test
vai rodar todos os testes  da classe Pagamento

mvn '-Dtest=br.com.uol.upbc.it.be.Pagamento#metodo*' test

vai rodar apenas o método especificado.
