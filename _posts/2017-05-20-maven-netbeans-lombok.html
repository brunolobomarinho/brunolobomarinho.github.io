It's not so easy to use the latest versions of lombok, netbeans and maven together, I found a good solution in this blog post where you have more details. But I'll give a simplified explanation here and make a sample project available in github.

1- Configure your pom. {% gist bdmstyle/5c1f13825314ca794688c517706b5526 %}

2 - At Project Properties->Actions->Activate Profiles and activate "lombok-build" to all operations you need like "build".

3- It works! Note that on command line mvn clean package is working normal (for non netbeans users like jenkins or eclipse users for example).
